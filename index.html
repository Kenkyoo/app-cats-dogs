Bienvenido a nuestra galería

Nuestra aplicación te permite explorar una amplia colección de imágenes de perros y gatos. Utiliza el interruptor para elegir entre perros y gatos, y descubre información detallada sobre cada raza.

¿Cómo utilizar esta aplicación?

Es muy fácil. Al iniciar, verás algunas imágenes destacadas. Puedes buscar razas específicas usando el buscador, o simplemente explorar las imágenes utilizando el botón de "Cargar más".

Funciones adicionales

Además de las imágenes, puedes encontrar información detallada sobre cada raza, incluyendo su origen, temperamento y enlaces para saber más.
